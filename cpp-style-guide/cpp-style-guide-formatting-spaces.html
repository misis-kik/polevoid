Расставляйте пробелы в соответсвии с рекомендациямиFalse

Note

Перечисление всех ситуаций, когда необходимо ставить или не ставить пробел, не является полным.

Ставьте пробел после ключевых слов языка перед открывающей круглой скобкойFalse

Hint

while (should_go) {
  // ...
}

Error

while (should_go) {
  // ...
}

Не ставьте пробел после имен функций перед перед открывающей круглой скобкойFalse

Hint

MoveNext()

Error

MoveNext ()

Не ставьте пробел между унарным оператором и его операндомFalse

Hint

++i

Hint

!IsGood()

Error

++ i

Error

! IsGood()

Ставьте пробел между бинарным/тернарным оператором и его операндамиFalse

Исключением является оператор запятая.

Hint

a = (b + c) * d;

a = (c < d) ? (a - c) : (d - a);

Error

a=(b+c)*d;

a = c<d ? a-c : d-a;

Ставьте пробел после запятой и не ставьте передFalse

Hint

DoSomething(a, b, c, d);

Error

DoSomething(a,b,c,d);

DoSomething(a , b , c , d);

Не ставьте пробел между базовым типом и ‘*’ или ‘&’ в производных типахFalse

Hint

T* p(&obj);

Hint

void CheckForErrors(const T& val);

Error

T *p(&obj);

Error

T * p(&obj);

Error

void replace( T& val);

Ствьте пробел между частями инструкции цикла forFalse

  • после точки с запятой в инструкции цикла for

Hint

for (int i(0); i < n_action; i += 1) {
  // ...
}

Error

for (int i(0) ; i < nAction ; i += 1) {
  // ...
}

Error

for (int i(0);(i < nAction);i += 1) {
  // ...
}
  • до и после двоеточия в интервальной версии инструкции for

Hint

for (auto page : pages) {
    page.close();
}

Error

for (auto page: pages) {
    page.close();
}

Не ставьте пробел после открывающей круглой скобки и перед закрывающейFalse

Hint

DoSomething(a, b, c, d);

Hint

if (is_ready)

Error

DoSomething( a, b );

Error

DoSomething(a , b , c , d);

Error

if ( isReady )
