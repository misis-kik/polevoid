3. Инсталляция SVN-клиент TortoiseSVN (Windows)False

Актуальную версию клиента можно скачать с официального сайта (с учетом версии и битности вашей ОС).

Процедура установкия клиента является стандартной для windows-приложений. Далее рассмотрим основные шаги по развертыванию SVN-клиента TortoiseSVN (на примере версии 1.10.1 для 64 битной операционной системы). После запуска инсталлятора появляется стартовой окно.

Стартовое окно инсталлятора TortoiseSVN.

Проверив возможность установки (доступного места и т.д.) инсталлятор разблокирует кнопку Next, по нажатию кототорой происходит переход к следующему экрану.

Окно лицензионного соглашения инсталлятора TortoiseSVN

После ознакомления с текстом соглашения, если вы с ним согласны и хотите продолжить установку клиента, нажите кнопку Next и перейдите к окну пользовательской настройки компонент инсталлятора.

Окно пользовательской настройки компонент инсталлятора TortoiseSVN.

При необходимости, вы можете настроить необходимый вам состав устанавливаемых компонент. Важно убедится, что среди устанавливаемых выбрана компонента command line client tools, что позволит в дальнейшем использовать консольный SVN-клиент.

Note

Если консольный клиент не был своевременно установлен, то его можно установить позже путем повторного запуска инсталлятора с выбором утилит коммандной строки среди устанавливаемых компонент в процессе инсталляции.

Уточните состав компонент и нажите кнопку Next.

Меню TortoiseSVN для неверсифицированных папок.

Нажмите кнопку Install.

Окно инсталлятора TortoiseSVN в процессе установки (стадия и прогресс).

Подождите, пока инсталлятор закончит свою работу.

Финальное окно инсталлятора TortoiseSVN.

Нажмите кнопку Finish.
