2. SVN-клиент TortoiseSVN (Windows)False

При необходимости скачивания и инсталляции клиента ознакомьтесь с соответсвующим разделом.

Проводник Windows (далее Проводник) — это графический интерфейс доступа к файловой системе в операционных системах семейства Microsoft Windows. Для desktop-систем Проводник является основой графической оболочки OC (пиктограммы рабочего стола, панель задач, меню «Пуск»).

SVN-клиент TortoiseSVN интегрируется с Проводником, то есть является расширением Проводника и не может быть запущен через пункт меню Пуск. После установки клиент добавляет новые пункты в контекстные меню Проводника, через которые осуществляется доступ к основному функционалу клиента (по нажатию правой кнопки мыши(.

Меню TortoiseSVN для неверсифицированных папок.

Меню TortoiseSVN для неверсифицированных папок.

Note

Язык отображения пунктов меню зависит от настроек системы.

Клиент изменяет отображение как самих версифицированных папок, так и объектов файловой системы (папок и файлов) внутри версифицированных папок.
