===============================================================================
Инсталляция SVN-клиент TortoiseSVN (Windows)
===============================================================================

Актуальную версию клиента можно |tortoisesvn_download_link| с официального сайта (с учетом 
версии и битности вашей ОС).

.. |tortoisesvn_download_link| raw:: html

   <a href="https://tortoisesvn.net/downloads.html" target="_blank">скачать</a>

Процедура установкия клиента является стандартной для windows-приложений. 
Далее рассмотрим основные шаги по развертыванию SVN-клиента TortoiseSVN (на 
примере версии 1.10.1 для 64 битной операционной системы). После запуска 
инсталлятора появляется стартовой окно.

.. figure:: images-vcs-svn-client-tortoisesvn/msi-step1.png
   :scale: 100 %
   :align: center
   :alt: 

   Стартовое окно инсталлятора TortoiseSVN.

Проверив возможность установки (доступного места и т.д.) инсталлятор 
разблокирует кнопку :guilabel:`Next`, по нажатию кототорой происходит 
переход к следующему экрану.

.. figure:: images-vcs-svn-client-tortoisesvn/msi-step2.png
   :align: center

   Окно лицензионного соглашения инсталлятора TortoiseSVN

После ознакомления с текстом соглашения, если вы с ним согласны и хотите 
продолжить установку клиента, нажите кнопку :guilabel:`Next` и перейдите
к окну пользовательской настройки компонент инсталлятора.

.. figure:: images-vcs-svn-client-tortoisesvn/msi-step3.png
   :align: center

   Окно пользовательской настройки компонент инсталлятора TortoiseSVN.

При необходимости, вы можете настроить необходимый вам состав 
устанавливаемых компонент. Важно убедится, что среди устанавливаемых 
выбрана компонента :guilabel:`command line client tools`, что позволит в 
дальнейшем использовать :ref:`консольный SVN-клиент <vcs-svn-client-svn>`.

.. note:: Если консольный клиент не был своевременно установлен, то его 
   можно установить позже путем повторного запуска инсталлятора с выбором 
   утилит коммандной строки среди устанавливаемых компонент в процессе 
   инсталляции.

Уточните состав компонент и нажите кнопку :guilabel:`Next`.

.. figure:: images-vcs-svn-client-tortoisesvn/msi-step4.png
   :align: center

   Меню TortoiseSVN для неверсифицированных папок.

Нажмите кнопку :guilabel:`Install`.

.. figure:: images-vcs-svn-client-tortoisesvn/msi-step5.png
   :align: center

   Окно инсталлятора TortoiseSVN в процессе установки (стадия и прогресс).

Подождите, пока инсталлятор закончит свою работу.

.. figure:: images-vcs-svn-client-tortoisesvn/msi-step6.png
   :align: center

   Финальное окно инсталлятора TortoiseSVN.

Нажмите кнопку :guilabel:`Finish`.
